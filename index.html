FBIS - Michigan Forest Biomass Information System

Interactive web application that allows the user to determine forest biomass volumes in a selected area. The application utilizes the ArcGIS Javascript API on the front end and a combination of ArcGIS maps, models and Python on the backend to serve data over a RESTful API.

FBIS Web Application



Sawmill Log Tally

This offline capable web application was developed for a local sawmill during late 2015. It allows the user to tally logs by species/grade/volume and upload the data to a server at the end of the shift. The server based upload process (PHP) updates inventory and reporting databases with data from this app and another embedded system.
